API & Développement

IncidentsRATP, pour pouvoir simplement contribuer et utiliser nos données, met à votre disposition une API REST permettant d'accéder aux informations dans les formats usuels JSON/XML.

Comment ?

C'est simple et voici quelques URL utilisables pour accéder à l'API :

Dernier incidents par minute/heure/jour

Il vous suffit d'accéder à l'URL suivante en replaçant (scope, format) parmis les valeurs suivantes. Pour le scope : (minutes,hour,day) et pour le format (json, xml) :

/api/incidents.format/scope
